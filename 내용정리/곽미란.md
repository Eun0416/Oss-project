1. 프로그래밍 기초 & 개발환경
■ 프로그래밍 언어

컴퓨터와 소통하기 위한 규칙

0과 1을 사람이 이해하기 쉽게 추상화한 표현 방식

■ JDK / JRE / JVM
구성	설명
JDK	개발 도구(컴파일러 포함), 자바 개발 시 필요
JRE	실행 환경(라이브러리 제공), 실행 시 필요
JVM	바이트코드 실행, OS 독립성 제공
■ 개발환경 구성

JDK 설치 → 환경 변수 설정

Eclipse/IntelliJ로 프로젝트 생성

■ 자바 소스 구조

패키지 → 클래스 → main()

주석: //, /* */, /** */

주요 키워드: public, int, class, new …

2. 변수와 데이터 타입
■ 변수

메모리 공간에 붙여진 이름

지역 변수는 반드시 초기화 필요

■ 변수 규칙

문자, 숫자, _, $ 사용 가능

숫자로 시작 불가

변수명: lowerCamelCase

클래스명: UpperCamelCase

예약어 사용 금지

■ 데이터 타입
✔ 기본 타입
분류	타입
정수	byte, short, int, long
실수	float, double
논리	boolean
문자	char
✔ 참조 타입

String, 배열, 클래스, 인터페이스 등

■ 문자열

" "로 표현

특수문자(escape): \n, \t, " , \

텍스트 블록 지원: """ ... """

■ 타입 변환

자동 변환: 작은 타입 → 큰 타입

강제 변환: (타입) 값

주의: byte/short/char → 연산 시 자동으로 int 승격

■ 입출력

Scanner 입력

print / println / printf 출력

3. 연산자
■ 연산자 종류

산술: + - * / %

비교: == != > < >= <=

논리: && || ! ^

비트: & | ^ ~ << >> >>>

증감: ++ --

대입: = += -= *= /=

삼항: 조건 ? A : B

■ 핵심 개념

실수 비교 시 정밀도 차이 주의

문자열 비교는 equals()

5/0.0 = Infinity, 5%0.0 = NaN

오버플로우/언더플로우 발생 가능

4. 제어문
■ 조건문
✔ if 계열

if → else if → else

조건식의 true/false에 따라 분기

✔ switch

case에 따라 분기

break 없으면 다음 case로 넘어감

문자열, 정수, 문자 모두 가능

switch 표현식: yield 사용 가능

■ 반복문
반복문	특징
for	반복 횟수 명확
while	조건이 true 동안 반복
do-while	최소 1번 실행
■ 흐름 제어

break: 반복 종료

continue: 다음 반복 진행

라벨 break: 중첩 반복 한 번에 종료

5. 문자열 · 배열 · JVM 메모리
■ JVM 메모리 구조
영역	내용

■ 문자열(String)

equals() → 실제 내용 비교

주요 메소드

length(), charAt(), substring()

indexOf(), replace(), split()

■ 배열(Array)

동일 타입 저장

길이 고정

복사

얕은 복사: 참조 공유

깊은 복사: 별도 메모리 생성

배열 도구:

System.arraycopy()

clone()

Arrays.copyOf()

■ enum

6. 객체지향 프로그래밍(OOP)
■ 절차지향 VS 객체지향
절차지향	객체지향
함수 중심	객체 중심
과정 강조	데이터 + 기능 묶음
■ OOP 핵심 4특징

캡슐화 : 정보 은닉

상속 : 코드 재사용

추상화 : 공통 특징만 모델링

다형성 : 동일 호출 → 다른 동작

■ 클래스 구성요소

필드: 속성

메소드: 동작

생성자: 초기화

■ static / final

static: 공용 멤버, 객체 없이 사용

final: 값 변경 불가

static final: 상수(대문자)

■ 패키지 / import

패키지: 코드 구조화

import: 외부 패키지 사용

■ Getter / Setter

private 필드 접근용 메소드

■ 싱글톤 패턴

프로그램 전체에서 단 하나의 객체만 존재하게 하는 방식
